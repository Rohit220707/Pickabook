# Pickabook AI Personalisation

A full-stack prototype that allows users to upload a child's photo and receive a personalized illustration.

## Tech Stack
- Frontend: ReactJS, Axios
- Backend: Node.js, Express

## How to Run

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start

## Notes
AI generation is mocked for stability. Architecture supports Instant-ID / SDXL integration in v2.
